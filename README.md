# X Lib

X Marks The Lib.


## Install

``` bash
# for node
npm install x-lib --save
```

## Build

``` bash
# install dependencies
$ npm install

# lint, test, clean dist rebuild
$ npm run dist

# only build (babel)
$ npm run build

# only lint (eslint)
$ npm run lint
```
